# Angular2-Demo


## 使用
- 确保 Node 版本为 4.0 及以上
- 执行 `npm install` 安装依赖包
- 执行 `npm start` 编译并启动本地server
- 打开浏览器访问 http://localhost:3000 

## 内容
1. 基于webpack的项目种子搭建
2. 内置指令的使用
3. 自定义指令的开发与使用
4. 服务的开发与使用
5. 父子组件双向通讯
